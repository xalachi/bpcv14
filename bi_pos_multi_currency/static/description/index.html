POS-Point of Sales Multi Currency pricelist Odoo Apps

If you are searching for an app od Odoo which will help you to payment with multi currency in POS-Point of sale screen.By default Odoo doesn't support payment on different currency on point of sale, after installing this odoo apps, you can able to do apply different currency's pricelist on point of sale screen and after apply pricelist products/order amount has been converted with selected pricelist in Odoo POS.
